# Product Positioning

## One-Line Pitch

swarm-flow is an AI-native software delivery engine that moves work from feature idea to shipped change using structured flows, specialist agents, engineering skills, hooks, connected tools, and policy gates.

## Distinction

swarm-flow is not a prompt pack, chat UI, or autonomous merge bot. It is a delivery state machine with skills and hooks plugged into it.

## Category

Governed AI software delivery orchestration.

## Primary Users

- engineering teams adopting coding agents
- platform teams standardizing AI delivery workflows
- technical leads who need auditability and approvals
- individual builders who want repeatable feature delivery

## Core Promise

Every meaningful step creates an artifact, every transition can be inspected, and every external write can be previewed and governed.
